HTML  - Hyper Text Markup Language

HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages. It defines the meaning and structure of content using elements and tags, which browsers interpret to display text, images, links, and other media. Hypertext refers to links connecting web pages, while markup is the annotation of content with tags like p, h1, a,0etc., to define its role and presentation.

CSS  - Cascading Style Sheets

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files
